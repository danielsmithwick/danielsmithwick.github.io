# Personal Portfolio Website

A clean, modern portfolio website built with HTML, CSS, and hosted on GitHub Pages.

## Features

- 📱 Fully responsive design
- 🎨 Modern gradient styling
- ⚡ Fast and lightweight
- 🔗 Smooth navigation
- 📱 Mobile-first approach

## Sections

- **About** - Tell visitors about yourself and your skills
- **Projects** - Showcase your best work
- **Contact** - Links to your social profiles and email

## Getting Started

### Local Development

1. Open `index.html` in a web browser to preview your site locally
2. Edit the HTML content in `index.html` to personalize your information
3. Customize colors and styles in `styles.css`

### Deployment on GitHub Pages

#### Option 1: Using `username.github.io` Repository

1. Create a new GitHub repository named `username.github.io` (replace `username` with your actual GitHub username)
2. Clone the repository to your computer
3. Copy all files from this project into the repository
4. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Initial portfolio setup"
   git push origin main
   ```
5. Your site will be live at `https://username.github.io`

#### Option 2: Using a Regular Repository

1. Create a new GitHub repository (e.g., `portfolio`, `website`, etc.)
2. Clone the repository
3. Copy all files from this project into the repository
4. Push to GitHub
5. Go to Settings → Pages → Source and select the branch where your files are
6. Your site will be live at `https://username.github.io/repository-name`

## Customization

### Update Personal Information

Edit `index.html` and replace:
- `Your Name` - Your actual name
- `your.email@example.com` - Your email address
- `yourusername` - Your GitHub profile URL
- `yourprofile` - Your LinkedIn profile
- `yourhandle` - Your Twitter/X handle
- Project descriptions and links

### Change Colors

In `styles.css`, look for these main color variables:
- `#667eea` - Primary purple
- `#764ba2` - Secondary purple
- `#2c3e50` - Dark text
- Modify the gradient in the `.hero` section

### Add More Projects

Copy a `project-card` div block and update the title and description
